Estruturas condicionais

Estruturas condicionais servem para alterar o fluxo de execução do meu algoritmo. Após um teste de condição, se a condição for verdadeira, o fluxo de execução do código interno à estrutural condicional executa. Caso Contrário, o código salta para o final da estrutura

if (condicao == verdadeira){ comandos executados quando a condição é verdadeira } else{ comandos executados quando a condição é falsa }
